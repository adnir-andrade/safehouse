# The Safehouse - Frontend

## Getting Started

### Prerequisites

- Expo (~> 51.0.14)
- React / Types (~> 18.2.45)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/adnir-andrade/safehouse.git
   ```

2. Navigate to the frontend directory.

3. Install dependencies using yarn:

   ```bash
   yarn install
   ```

4. Start the Expo Server:

   ```bash
   yarn start
   ```

5. Use it!
